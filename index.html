Tidos de Dados e Variáveis

Aula 02

Anotações:

Comentários em JS:

"//" Para comentários em 1 linha.

"/**/" Para comentários com mais de 1 linha.



Variáveis servem para receber dados.



Declarando uma variável:

var carro = 'carro1' (leia: variável "carro" recebe o valor "carro1")

No JavaScript moderno, além de usar a palavra "var", também podemos usar a palavra "let".

Declarando variáveis do tipo String:

var s1 = 'JavaScript'

var s2 = "Programação"

var s3 = `Igor`



Regras para identificadores de variáveis:

Podem começar com letras, $ ou _

Não podem começar com números

É possível usar letras ou números

É possível usar acentos e símbolos

Não podem conter espaços

Não podem ser palavras reservadas (ex: "var", "function", etc...)



Principais tipos de dados:

number: 5, 7, -12, 0.5, -15.34

string: "Google", 'JavaScript', `3245-567`

boolean: true e false



Data types:

number: Infinity / NaN (not a number)

string

boolean

null

undefined

object: Array

function



Comando para identificar o tipo de dado:

typeof (ex: typeof 20 --> number, typeof false --> boolean)

Manipulação de Dados

Aula 03

Anotações:

Guardando e usando dados de entrada em uma variável:

var nome = window.prompt('Digite seu nome:')

window.alert('Olá, ' + nome + '!') <-- concatenação

*o "window.prompt" retorna uma string



Convertendo string para número:

Comandos: Number.parseInt(n) ou parseInt(n) / Number.parseFloat(n) ou parseFloat(n) / Number(n)

parseInt() ---> numero inteiro

parseFloat() ---> numero real

Number() ---> numero inteiro e inteiro

Exemplo:

var n1 = Number(prompt('Digite um número'))

var n2 = Number(prompt('Digite mais um número'))

var res = n1 + n2

window.alert('A soma dos números é: ' + res)



Convertendo número para string:

Comandos: String(n) / n.toString()



Formatando strings:

var s= 'JavaScript'

"Estou aprendendo + s" // não faz interpolação

'Estou aprendendo' + s // faz concatenação

`Estou aprendendo ${s}` // template string (com placeholder -> ${})

s.length --> // quantos caracteres (letras, símbolos ou números) a string tem

s.toUpperCase() --> // tudo para "MAIÚSCULAS"

s.toLowerCase() --> // tudo para "minúsculas"



Formatando números

var n1 = 1543.5

n1.toFixed(2) --> 1543.50

String(n1).replace('.', ',') --> 1543,5

* para trocar o ponto de um número para uma vírgula, é preciso manipular o tipo da variável para string

n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) --> R$1.543,50

n1.toLocaleString('pt-BR', {style: 'currency', currency: 'USD'}) --> US$1.543,50

Aula 04

Anotações:

Alguns Operadores no JS:

Aritiméticos / Atribuição

/ Relacionais / Lógicos

/ Ternário


Operadores estudados nesta aula:

Aritiméticos / Atribuição


Operadores aritiméticos:

5 + 2 -> 7

5 - 2 -> 3

5 * 2 -> 10

5 / 2 -> 2.5 (divisão real)

5 % 2 -> 1 (divisão inteira)

5 ** 2 -> 25


Ordem de precedência de operadores:

1º- "()"

2º- "**"

3º- "*" "/" "%"

4º- "+" "-"


Operador de atribuição simbles:

= -> recebe

Ex: var a = 5 + 3 -> var a recebe 5 mais 3


Auto-atribuição:

var n = 3

n = n + 4

n -> 7


Simplificando auto-atribuição:

var n = 3

n = n + 4 simplificando -> n+=4


Operadores de incremento/decremento:

var x = 5

x = x + 1 -> x += 1 -> x++

x = x - 1 -> x -= 1 -> x--

Pré-incremento/decremento: ++x ou --X

Pós-incremento/decremento: x++ ou x--

Anotações:

Operadores relacionais:

> Maior que

< Menor que

>= Maior ou igual

<= Menor ou igual

== Igual

!= Diferente


*O resultado de uma expressão com operador relacional, vai ser sempre um valor booleano.

5 > 2 --> true

7 < 4 --> false

8 >= 8 --> true

9 <= 7 --> false

5 == 5 --> true

4 != 4 --> false


5 == 5 --> true

5 == '5' --> true

*O sinal "==" não testa o tipo da variável.

Operadores de Identidade:

=== --> Idêntico

!== --> Desigual restrito

5 === '5' --> false

5 === 5 --> true


Operadores lógicos:

! --> negação

&& --> conjunção (and)

|| --> disjunção (or)

Negação:

false --> true
true --> false

Conjunção:

true true --> true
true false --> false
false true --> false
false false --> false

Disjunção:

true true --> true

false true --> true
false false --> false
