Tidos de Dados e Variáveis

Aula 02

Anotações:

Comentários em JS:

"//" Para comentários em 1 linha.

"/**/" Para comentários com mais de 1 linha.



Variáveis servem para receber dados.



Declarando uma variável:

var carro = 'carro1' (leia: variável "carro" recebe o valor "carro1")

No JavaScript moderno, além de usar a palavra "var", também podemos usar a palavra "let".

Declarando variáveis do tipo String:

var s1 = 'JavaScript'

var s2 = "Programação"

var s3 = `Igor`



Regras para identificadores de variáveis:

Podem começar com letras, $ ou _

Não podem começar com números

É possível usar letras ou números

É possível usar acentos e símbolos

Não podem conter espaços

Não podem ser palavras reservadas (ex: "var", "function", etc...)



Principais tipos de dados:

number: 5, 7, -12, 0.5, -15.34

string: "Google", 'JavaScript', `3245-567`

boolean: true e false



Data types:

number: Infinity / NaN (not a number)

string

boolean

null

undefined

object: Array

function



Comando para identificar o tipo de dado:

typeof (ex: typeof 20 --> number, typeof false --> boolean)
